# NCKU little den den
NCKU freshman camp

## Usage

- Install
```
  ./func init
```

- Use Npm
```
  ./func npm (argument)
```

- Use Bower
```
  ./func bower (argument)
```

- Transform js from jsx:
```
  ./func jsx
```
